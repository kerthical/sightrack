# ⚡ Quick Start

## Requirements

- Python 3.9

## Setup

```bash
git clone https://github.com/kerthical/sightrack.git
cd sightrack
python runner.py setup
```

## Run

```bash
python runner.py run <camera_id/video_path> [--headless] [--output <output_path>]
```
